# Fuel Cost and Travel Time Calculator

A C# Windows Forms application that calculates estimated fuel cost and travel time based on user input.

## Features
- Input validation using TryParse
- Protection against division by zero
- Fuel cost calculation using methods
- Travel time calculation using methods
- Currency formatting in USD
- Clear and Exit confirmation functionality
- Prevents Exception try/catch


## Technologies Used
- C#
- .NET Windows Forms
- Visual Studio

## How It Works
1. User enters distance, fuel consumption, fuel price, and speed
2. Application validates all inputs
3. Fuel cost and travel time are calculated
4. Results are displayed in a user-friendly format
